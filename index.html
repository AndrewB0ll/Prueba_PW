Introducción

Mi nombre es Carlos Andrés Bolaños Ortega, soy estudiante de Tecnología en Electrónica Industrial en la Facultad de Ingeniería, actualmente cursando mi 7mo semestre.

Este proyecto nace de mi pasión por la electrónica y el desarrollo de soluciones innovadoras que combinen hardware y software para resolver problemas cotidianos. Con este sistema IoT para motocicletas, busco aplicar los conocimientos adquiridos durante mi formación académica, integrando tecnologías modernas como el ESP32 y protocolos de comunicación IoT para ofrecer una solución eficiente y de bajo costo.

El proyecto consiste en el desarrollo de un sistema IoT para motocicletas que permita monitorear el estado de las luces, implementar un cortacorriente para mayor seguridad, y ofrecer la funcionalidad de encendido remoto.

El sistema está diseñado para ser fácil de usar y altamente eficiente, integrando tecnologías modernas como el ESP32 y protocolos de comunicación IoT.

Problemática

Fallas en el sistema de luces

Muchas veces, los usuarios no se dan cuenta de que las luces de su moto no funcionan correctamente, lo que puede generar multas o accidentes.

Vulnerabilidad a robos

Las motocicletas son vulnerables a robos debido a la falta de sistemas de seguridad avanzados.

Falta de comodidad

El encendido manual de la moto puede ser incómodo, especialmente en climas fríos.

Solución Propuesta

El sistema propuesto integra tres funcionalidades principales:

  • Monitoreo de luces en tiempo real.
  • Cortacorriente activado de manera remota.
  • Encendido remoto de la moto.

Tecnologías Utilizadas

ESP32

Microcontrolador con Wi-Fi y Bluetooth para la comunicación IoT.

Sensores de Corriente

Para detectar el estado de las luces.

Relés

Para controlar el cortacorriente y el encendido remoto.

Funcionamiento del Sistema

  1. Los sensores detectan el estado de las luces y envían datos al ESP32.
  2. El ESP32 procesa los datos y los envía a la aplicación web.
  3. El usuario puede activar/desactivar el cortacorriente o encender la moto desde la aplicación web.

Aplicación Web

La aplicación web permitirá al usuario:

  • Ver el estado de las luces en tiempo real.
  • Activar/desactivar el cortacorriente.
  • Encender la moto de manera remota.

Beneficios del Proyecto

Seguridad

Reducción del riesgo de robos gracias al cortacorriente.

Comodidad

Encendido remoto y monitoreo en tiempo real.

Prevención

Detección temprana de fallos en las luces.
